## Getting Started

### Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (version 14 or higher)
- [Expo CLI](https://docs.expo.dev/get-started/installation/) (for easier development with React Native)

### Installation

1. Clone the repository:

- git clone https://github.com/yourusername/mobile-wallet.git
- cd mobile-wallet
- npm install
- expo start
- Follow the instructions in the terminal to run the app on an Android or iOS emulator or your physical device.
